# Experiment 38 SVG Test

This is a site_data display test for an SVG-rendered FDRS Experiment 38 atlas.

The source wrapper runs the canonical HiR Sage experiment and changes only the
output format from PNG to SVG. The generated atlas is intended to test how SVG
graph artifacts display through the site_data manifest.

Generated variants:

- `a=72`, `B=60`: `fdrs_exp38_kernel_cylinder_radix_atlas_a72_B60.svg`
- `a=210`, `B=462`: `fdrs_exp38_kernel_cylinder_radix_atlas_a210_B462.svg`
